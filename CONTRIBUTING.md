# Contribuir a REI

Gracias por querer contribuir a REI.

Nuestro objetivo es democratizar el acceso al conocimiento mediante una infraestructura educativa local, abierta y fácil de implementar.

Puedes contribuir de diferentes maneras:

- Corrección de errores.
- Documentación técnica.
- Recursos educativos.
- Traducciones.
- Optimización del hardware.
- Mejoras en la interfaz web.
- Pruebas del sistema.

Antes de realizar cambios importantes, crea un Issue para discutir la propuesta con el equipo.

Toda contribución debe mantener el enfoque educativo y el propósito social del proyecto.
